Flutter 是 Google 公司推出的一个 UI 框架，可以快速开发出适用于 Android 和 iOS 的 APP。也是将来 Fuchsia 操作系统采用的 UI 框架，配合 dart 语言。
笔者打算用 flutter 开发一个简单的 demo，试试 flutter。

执行完，可以查看相关工具情况。笔者平时开发使用 intelij idea，给它装好 flutter、dart 插件就可以开发了。

设计

自己开发就没那么多流程了，什么 UE，UI，不存在的。
想了想，打算做一个记账 APP，就叫阿财吧！图标就用微信的头像，一只萌萌的柴犬。
功能包括，首页展示账单列表，新增，修改，删除，上传图片，下载图片，以及登录注册。

接口

根据功能，我们开发了几个后端接口。

  • http://localhost:8080/api/oauth/publicKey GET 获取公钥
  • http://localhost:8080/api/oauth/signUp POST 注册
  • http://localhost:8080/api/oauth/signIn POST 登录
  • http://localhost:8080/api/classification GET 查询所有支出（收入）分类
  • http://localhost:8080/api/moneyRecord GET 查询所有账单记录
  • http://localhost:8080/api/moneyRecord/:id GET 查询单条账单记录
  • http://localhost:8080/api/moneyRecord POST 新增账单记录
  • http://localhost:8080/api/moneyRecord/:id PUT 修改单条账单记录
  • http://localhost:8080/api/moneyRecord/:id DELETE 删除单条账单记录
  • http://localhost:8080/api/moneyRecord/upload POST 上传图片
  • http://localhost:8080/api/moneyRecord/download GET 下载图片
