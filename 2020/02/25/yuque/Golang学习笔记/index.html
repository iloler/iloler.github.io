基础

Hello World

包

package main
程序从 main 包开始运行。
import "fmt"
导入 fmt 包，如果需要导入多个包，可以用括号括起来。

函数

函数使用关键字func关键字，可以有零个或多个参数，也可以有零个或多个返回值。比较特别的，还有个命名返回值。

基础数据类型

比较特别是 go 提供了复数形式的数据类型。
int, uint 和 uintptr 在 32 位系统上通常为 32 位宽，在 64 位系统上则为 64 位宽。

变量

声明变量可以用var关键字，可以出现在包级别和函数级别。
也可以使用:=这种简短的声明方式，但是这种方式不能在函数外使用。

从代码运行结果也能看出来，基础数据类型的零值。
数值类型为 0，
布尔类型为 false，
字符串为 “”（空字符串）。

类型转换与推导

go 转换数据类型，需要显示声明。如果上面的代码，去掉了float64(a)中的 float64()则会报错。
cannot use a (type int) as type float64 in assignment

声明变量不指定类型的话，会根据等号右边的指进行推导。

常量

常量生命和变量类似，使用关键字const，不过常量不能使用:=。

流程控制语句

if

go 的if语句，不需要()，但是需要{}。if 语句可以在条件表达式前执行一个简单的语句。
该语句声明的变量作用域仅在 if 之内。

for

和 if 一样，条件语句不需要()。初始化语句和后置语句是可选的。

go 中循环只有for可以使用，没有while这个关键字。

如果省略了条件语句的话，可以当成无限循环。

switch

运行结果：
真的是 2
真的是 3
完事儿～
和其他语言对比，go 在case执行完会自动终止，除非使用fallthrough关键字。
switch没有条件时，和switch true一样。

defer

运行结果：
hello
哈哈哈
world
defer 函数会在外层函数执行完，再执行。推迟的函数调用会被压入一个栈中。当外层函数返回时，被推迟的函数会按照后进先出的顺序调用。

更多类型

指针

指针保存了值的内存地址,类型 *T 是指向 T 类型值的指针。其零值为 nil。& 操作符会生成一个指向其操作数的指针。

运行结果：
1
0xc00005ef30
1
666
666

结构体

一个结构体是由一组字段组成，可以使用“.”来访问这些字段。也可以通过结构体指针访问。

运行结果：
阿财
{tony 18}
tony
{hello 18}
&{hello 18}

数组

类型 [n]T 表示拥有 n 个 T 类型的值的数组。

切片

数组大小有限制，切片是 go 提供一种灵活的数据结构。可以通过关键字make创建，append关键字添加元素，range关键字迭代。

映射

map 是一个 key,value 集合，通过关键字make创建，通过 key 获取 value，通过delete删除值。

方法和接口

方法

go 没有类的概念，不过有结构体。方法即函数，不过需要在func和方法名之间加上参数接收者。
参数接收者还可以是指针，如果是值接收者，那么函数内会拷贝一份数据，进行操作，不会改变原来的值。

接口

接口是由一组方法签名定义的集合。这一点和 java 差别很大，不需要 implements 关键字去实现。只要实现了interface定义的所有方法，就可以认为实现了这个接口。
比较特殊的是空接口，interface{}当 Java 里的 object 用吧。

类型断言

运行结果：
hello true
0 false
panic: interface conversion: interface {} is string, not float64

类型选择

类型选择可以用 x.(type)来判断。

并发

协程

goroutine是 go 的一个大杀器。函数前面加一个 go 就可以欢快滴跑起来了。

信道

channel 是有类型的管道，可以通过<-来传输或者接收数据。箭头就是数据的流向。channel 需要通过 make 创建，可以设置第二个参数作为缓冲区。ch := make(chan int, 100)
发送者可通过 close 关闭一个信道来表示没有需要发送的值了，接收者可以通过第二个参数判断信道是够关闭了。
循环 for i := range c 会不断从信道接收值，直到它被关闭。
select 语句使一个 Go 程可以等待多个通信操作。
select 会阻塞到某个分支可以继续执行为止，这时就会执行该分支。当多个分支都准备好时会随机选择一个执行。
结合 switch，当所有分支都没准备好，执行 default 分支。

同步

Go 标准库提供了互斥锁sync.Mutex来处理同步的问题，可以用defer来保证锁最终会被释放。
